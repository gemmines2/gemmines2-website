How to Submit to Google Merchant Center
  1. Download feed.xml using the button above
  2. Upload to GitHub: Go to your repo → click feed.xml → pencil ✏️ → select all → paste XML content → commit
  3. Your feed URL will be: https://gemmines2.github.io/gemmines2-website/feed.xml
  4. Go to Google Merchant Center → Products → Feeds → click + to add new feed
  5. Select: Country = your target country, Language = English, Feed name = Gemmines2
  6. Choose: "Scheduled fetch" → paste your feed URL → set fetch frequency to Daily
  7. Click Fetch Now to test immediately — Google will show any errors
  8. For CSV: alternatively use Products → Feeds → Upload → select the CSV file directly
