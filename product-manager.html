🔒 How to Avoid Corruption & Stabilize Your Site

⚠️ Root cause of past corruption: Opening different copies of this tool (Desktop file vs GitHub URL) creates separate, disconnected data. ALWAYS open this tool the same way — via its GitHub Pages URL — never from a Desktop copy.
  1. Always open this tool via its GitHub URL
    e.g. https://gemmines2.github.io/gemmines2-website/admin.html — bookmark it.
  2. Use Single Item Snippet for new products
    Go to "Generate JS File" → "Single Item Snippet".
  3. Use Full JS File only when doing a big cleanup
    Replaces everything cleanly.
  4. For images and video — everything uploads to the SAME images/ folder
    This site stores both photos and videos together in the images/ folder on GitHub. Click "Upload Images" or "Upload Video" and select your files — the correct path is filled in automatically.
  5. Wait 2 minutes after GitHub commit
    GitHub Pages takes 1–3 minutes to publish.
  6. Duplicate ID protection is on
    The tool already warns you if you try to reuse an existing Product ID.
