Return Window: 14 Days | Return Cost: Free | Restocking Fee: $0

At Gemmines2, customer satisfaction is our top priority. We offer a transparent return process for our natural gemstones.

14-DAY RETURN WINDOW

  • We offer a 14-day return policy.
  • Return requests must be initiated within 14 days of the confirmed delivery date.
  • Items must be returned in their original packaging, unused, and with all original certifications.

FEES & SHIPPING

  • Gemmines2 charges $0 restocking fees.
  • Return shipping for items that are defective or incorrect is Free.
  • For standard returns, the customer is responsible for safe, insured return shipping.

REFUND PROCESS

  • Once the returned item is received and inspected, a refund will be processed within 7 days.
  • Refunds are issued to the original payment method only.
